# Chit-Chat
## Introduction
<br/>
&emsp; Developed a realtime chat application using socket.io, where user can send message to a group of people who are connected to the server and also receive message from them.
<br/><br/>

## Features

  * User can converse with a group of people
  * Time will be mentioned in every messages
  * If any user joined the chat that will be shown in the interface.  Ex- Ram joined the chat
  * If any user left the chat then that will be shown in the interface.  Ex- Ram left the chat
  * If any online user is typing then that will be shown in the interface.  Ex- Ram is typing…
  * User can see the online users after clicking the Love Icon
  * User can also enable dark mode to chat in ease at night
  
<br/> <br/>
## Tools & technologies used:
  **Front End Technologies:** &nbsp; HTML, CSS, JavaScript <br/>
  **Back End Technologies:** &nbsp; NodeJS, ExpressJS, Socket.IO
